# 문의 모달 정의서

## 트리거

- `Header.tsx`의 "문의하기" 버튼 클릭 시 모달 open
- 기존 `<a href="mailto:...">` → `<button onClick>` 으로 변경

│     │  문의하기            [ × ] │       │
│     │───────────────────────────│       │
│     │  이름                      │       │
│     │  ┌───────────────────┐    │       │
│     │  │                   │    │       │

## 색상 테마

사이트 전역 테마를 그대로 따른다.

| 요소 | 색상 | Tailwind 클래스 |
|------|------|-----------------|
| 오버레이 배경 | 검정 70% 불투명 | `bg-black/70` |
| 모달 배경 | brand-black `#000000` | `bg-brand-black` |
| 모달 테두리 | brand-yellow `#FFD600` | `border border-brand-yellow` |
| 제목 텍스트 | brand-yellow | `text-brand-yellow` |
| 라벨 텍스트 | brand-yellow | `text-brand-yellow` |
| 입력 필드 배경 | 투명 | `bg-transparent` |
| 입력 필드 테두리 | brand-yellow | `border border-brand-yellow` |
| 입력 필드 텍스트 | 흰색 | `text-white` |
| 입력 placeholder | 회색 | `placeholder:text-neutral-500` |
| 닫기(×) 버튼 | brand-yellow | `text-brand-yellow` |
| 제출 버튼 (기본) | 노란 테두리 + 노란 텍스트 | `border border-brand-yellow text-brand-yellow` |
| 제출 버튼 (호버) | 노란 배경 + 검정 텍스트 | `hover:bg-brand-yellow hover:text-brand-black` |

## 폰트

- Pretendard Variable (프로젝트 전역 폰트, 별도 지정 불필요)

## 동작

| 동작 | 설명 |
|------|------|
| 열기 | Header "문의하기" 버튼 클릭 |
| 닫기 | × 버튼 클릭, 오버레이 클릭, `Esc` 키 |
| 스크롤 잠금 | 모달 open 시 `body` 스크롤 비활성화 |
| 포커스 트랩 | 모달 내부에서만 Tab 이동 |

## 폼 필드

| 필드 | HTML 타입 | 필수 | 추가 속성 |
|------|-----------|------|-----------|
| 이름 | `<input type="text">` | O | `maxLength={50}` |
| 이메일 | `<input type="email">` | O | |
| 문의 내용 | `<textarea>` | O | `rows={5}` |

## 상태 처리

| 상태 | UI 표현 |
|------|---------|
| 기본 | 폼 활성화, 제출 버튼 "제출하기" |
| 전송 중 | 폼 비활성화, 버튼 "전송 중..." |
| 성공 | 성공 메시지 표시 후 모달 닫기 |
| 실패 | 에러 메시지 표시, 폼 재활성화 |
